# Recruitment task

## How to start?

1. Install dependencies

```bash
$ yarn
```

2. Create your .env file, copy env variables from .env.example and complete it with your data

3. Run the app

```bash
$ yarn dev
```

## Env configuration

```
API_KEY=string required (get it from https://newsapi.org/)
BASE_URL=string optional (default "/")
```
